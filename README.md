# Sistema Quality Cars (acadêmico)

Sobre o Projeto

Este projeto foi desenvolvido em C como parte de um trabalho acadêmico.
O sistema simula um cadastro de veículos, permitindo operações básicas de CRUD em um arquivo de texto. Além disso, o programa inclui um aviso de privacidade reforçando boas práticas no tratamento de informações.

Funcionalidades: 

O sistema oferece as seguintes opções:

*Login com autenticação (usuário e senha, até 3 tentativas)
*Cadastrar veículo (marca, modelo, ano, placa e cor)
*Listar veículos cadastrados
*Editar veículo existente (busca pela placa)
*Excluir veículo (busca pela placa)
*Encerrar o programa com mensagem final personalizada

Estrutura do Projeto:

O repositório contém os seguintes arquivos:

Principal.cpp → Arquivo principal com o menu e fluxo do programa
funcoes.h → Cabeçalho com definições de estruturas e funções
funcoes.c → Implementação das funções (login, salvar, ler, editar, excluir)

Aviso de Privacidade

O sistema realiza apenas o tratamento mínimo necessário de informações pessoais, utilizadas exclusivamente para:

*Autenticação de usuários
*Registro interno de operações
*Garantia de integridade e segurança
*Nenhum dado é compartilhado com terceiros.
